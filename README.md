# 🌸 Flower Classifier

An end-to-end image classification system using ConvNeXtV2 and Hugging Face Transformers.

## Features
- Fine-tuned ConvNeXtV2 model
- Batch prediction script
- Gradio web interface
- Hugging Face compatible

## Run Locally

Create virtual environment:

python -m venv venv
venv\Scripts\activate

Install dependencies:

pip install -r requirements.txt

Run app:

python app.py
